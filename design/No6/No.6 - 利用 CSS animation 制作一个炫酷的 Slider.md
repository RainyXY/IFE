# No.6 - 利用 CSS animation 制作一个炫酷的 Slider
### 任务目的
- 学习CSS animation keyframes
- 理解动画中 关键帧的概念
- 熟悉并掌握 CSS3 动画属性

- 观看视频 demo ，通过 CSS animation 实现一个 全屏 Image Slider 的效果；
